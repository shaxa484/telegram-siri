# Telegram Siri Setup Wizard

Beautiful visual setup guide for Telegram Siri Integration.

## For Users

### How to Use

1. **Download** the DMG file from [GitHub Releases](https://github.com/shaxa484/telegram-siri/releases)
2. **Open** the DMG file
3. **Drag** the app to your Applications folder
4. **Right-click** the app and select "Open" (first time only)
5. **Follow** the step-by-step wizard!

That's it! The app will guide you through the entire setup process.
